HTML

  • HTML stands for hyper text markup language.
  • HTML gives content struture.
  • The title element defines the title for the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles assoiciated with the webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as character set, description, keywords, author, and viewport.
  • The script element embeds client-side scrips in the HTML document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations willl apply.
  • The image selector assigns property values to any image in the makup.
  • The display property is assigned with a block value, which assigns each image to start on a new line.
  • A link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location(URL) of the external resource.
  • The period preceding card designates this as a selector. Any HTML elements assigned to the class, card, will gain these property assignments form the CSS.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and swithces to it.

JavaScript

  • variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • In Javascript to declare a variable we use the var keyword.
  • All strings are surounded by quotation marks.
  • Each line of Javascript is ended with a semicolon.
  • Array is a single variable that used to hold a group of data that is usually related in some way.
  • An index number is a unique identification number given to different pieces of data within an array.
  • Arrays are zero-indexed and sequential, meaning that the first item begins with zero and increases by one.
